# kinoko-words

## Usage

If you are a user in China, you can use the Chinese npm mirror. 

```shell
# tabao npm mirror
pnpm config set registry https://registry.npmmirror.com/
# offical mirror
pnpm config set registry https://registry.npmjs.org/
```

Install project dependencies.

```shell
pnpm install
```
